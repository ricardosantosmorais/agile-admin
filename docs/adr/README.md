# ADRs do Admin v2

## Objetivo
Esta pasta registra as Architectural Decision Records do `admin-v2-web`.

Cada ADR descreve:
- contexto;
- decisao tomada;
- consequencias tecnicas;
- impacto no projeto.

ADRs nao substituem a documentacao geral. Eles existem para registrar decisoes que moldam a arquitetura e que devem continuar validas ao longo da evolucao do sistema.
